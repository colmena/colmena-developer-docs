## Download Colmena.

Clone Colmena in your workspace to the `colmena` folder.

```bash
cd ~/workspace/
git clone https://github.com/colmena/colmena.git colmena
```

## Install dependencies

Move into the `colmena` folder and run `npm install`

```bash
cd colmena
npm install
```
